# Proyecto

Este proyecto tiene un **backend** en Python y un **frontend** en Vite (React).

---

## Backend

1. Navegar a la carpeta `services`:

```bash
cd services
```
2.	Crear un entorno virtual:
```bash
mkdir venv
cd venv
python -m venv .
```
3.	Activar el entorno virtual:

En Linux/macOS:
source venv/bin/activate

4.	Instalar dependencias (si existe requirements.txt):

pip install -r requirements.txt

5.	Configurar el archivo .env dentro de la carpeta services con las variables necesarias para el backend.

python app.py


## Frontend
1.	Configurar el archivo .env del frontend (ubicado en la raíz del proyecto o en la carpeta de Vite) con la URL del API y otras variables necesarias.
2.	Instalar dependencias:
   yarn


3.	Correr la aplicación:
   yarn dev
